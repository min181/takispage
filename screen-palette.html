Screen Palette

A color and layout preview app for checking UI colors, rectangles, sizes, and corner radius on a real iPhone screen.

Overview

Screen Palette is designed for independent iOS developers who want to quickly test how background colors, shape colors, spacing, and rounded corners feel on an actual iPhone.

It is not intended to replace full design tools. Instead, it focuses on fast, practical visual checks from the device in your hand.

Features

  • Full-screen canvas for color and shape previews
  • Background color editing with a color picker and HEX input
  • Multiple editable rectangles with position, size, color, and corner radius controls
  • Immediate visual feedback while editing colors and shapes
  • Local palette storage on the device
